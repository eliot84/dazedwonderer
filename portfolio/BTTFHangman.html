Back to the Future - Hangman

Description: This is a study on event listeners that trigger actions when the computer keyboard keys are pressed. The project uses HTML, CSS, Javascript and HTML5 Sound to create a simple hangman game. The theme in this project comes from the movie Back to the Future.
